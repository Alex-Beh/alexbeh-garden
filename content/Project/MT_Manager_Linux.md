# Installation of MT Manager on Linux

1. Download the software from official website & extract

   ```
   tar -xvzf MT_Software_Suite_linux-x64_2019.3.2.tar.gz
   ```
2. sudo ./mtsdk_linux-x64_2019.3.2.sh
